Recon API

Recon API began as a passion project trying to ease the pains of communication between indie app developers and frustrated users. Its intent was to convert negative reviews on Google Play (and eventually, the App Store) into meaningful insight collected via integration with Recon API.

It was built over a weekend and launched the following Monday. It was used in almost all of the Android applications published by Thinglet, and saw some traction in the Android developer community.

Hightlights

  • Sourced and helped produce the Recon API intro video.
  • Learned (barely) enough Java in one weekend to throw together the Android SDK.
